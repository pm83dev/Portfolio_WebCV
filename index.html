Da Excel al controllo digitale della produzione

From Excel to digital production control

Un'azienda di produzione saponi gestiva tutto manualmente: ordini trascritti a mano, dati letti fisicamente in reparto, zero visibilità da remoto. Ho sviluppato un sistema integrato che connette il gestionale, il PLC e lo SCADA in un'unica interfaccia web.

A soap manufacturing company managed everything manually: hand-transcribed orders, on-site data reading, zero remote visibility. I built an integrated system connecting ERP, PLC and SCADA in a single web interface.

Dettaglio tecnico chiave: l'applicazione web è integrata direttamente nello SCADA Siemens come finestra embedded — l'operatore gestisce tutto da un'unica schermata.

Key technical detail: the web app is embedded directly inside the Siemens SCADA as a web window — operators manage everything from a single screen.

Settore Sector Manifatturiero — produzione saponi Manufacturing — soap production

Risultato Result Zero trascrizioni manuali, visibilità real-time, ufficio e reparto connessi Zero manual transcription, real-time visibility, office and shop floor connected
